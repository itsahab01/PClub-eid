# Eid Greeting Generator

A web-based application developed for the Programming Club to create personalized Eid greeting cards.

## Project Overview

This project allows users to enter their names and generate customized Eid greeting cards. The application automatically places the entered name onto a pre-designed greeting card and enables users to download the final image.

## Features

- Personalized Eid greeting card generation
- Dynamic name insertion
- Downloadable greeting card image
- Simple and user-friendly interface
- Responsive design
